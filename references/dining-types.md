# 4 类选店维度（美食 Skill 独家）

> 美食 Skill 与旅游 Skill 最大不同——选店维度决定内容调性。本文档解释 4 类选店策略的使用规则。

---

## 🏛️ 1. 老字号（百年传承）

**适合**: 文化深度 / 游客必打卡 / 怀旧

**典型特征**：
- 年龄 ≥ 50 年（建国前开店）
- 有非遗认证 / 老字号牌匾
- 服务员年纪 ≥ 50
- 价格偏中上

**内容调性**：仪式感 / 怀旧 / 游客视角
- 文案关键词：「百年」「传承」「非遗」「老师傅」
- 拍照关键词：雕花门窗 / 老师傅动作 / 老招牌

**风险提示**：
- 价格虚高（部分游客套餐）
- 排队 1-2h（必须早到）
- 高峰期可能扩大化经营（品质可能下降）

---

## 📸 2. 网红 KOL（出片神器）

**适合**: 24-30 岁的出片族 / 探店分享

**典型特征**：
- 装修网红风（霓虹 / 星空 / 二次元）
- 套餐营销（小红书套餐）
- 排队 2-4h 是常态
- 价格中等

**内容调性**：视觉冲击 / 朋友聚会 / 拍照出片
- 文案关键词：「出片」「排队也值」「必拍」
- 拍照关键词：招牌霓虹 / 菜品俯拍 / 干冰效果

**风险提示**：
- **价高质次常见**（探店要谨慎）
- 一阵风过去就「探雷」，生命周期短
- 排队党心理情绪大：好吃不好吃都写
- **强提示要踩店 2 次再推荐**

---

## 🪰 3. 苍蝇馆（本地人私藏）

**适合**: 本地人真正日常 / 反差感内容 / 烟火气

**典型特征**：
- 店面 ≤ 5 桌
- 无装修 / 无空调 / 无菜单
- 价格极低（人均 30-80）
- 没点评 / 没评论

**内容调性**：反差萌 / 烟火气 / 「比抖音爆火的 X 好吃多了」
- 文案关键词：「街角」「老客人」「3 代人」
- 拍照关键词：油渍桌 / 手写菜单 / 老板娘笑脸

**风险提示**：
- 环境堪忧（卫生问题）
- 游客难找（需本地人带路）
- **要强调"卫生可接受"** — 否则差评

---

## 🏮 4. 私房菜（高端 / 仪式）

**适合**: 商务宴请 / 庆祝场合 / 老饕

**典型特征**：
- 需提前 1-2 周预约
- 无固定菜单（时令定制）
- 环境独立空间
- 价格（人均 300-2000+）

**内容调性**：高端 / 仪式感 / 老饕视角
- 文案关键词：「私房」「时令」「独家」「非遗」
- 拍照关键词：盘器 / 时令食材 / 仪式感

**风险提示**：
- **强预订要求**（不能现场去）
- 高客单价 → 需要"非小红书套餐"
- 越私密 → 越不适合拍老板/客人

---

## 🎯 选店策略决策树

```
我该选什么店型？
│
├─ 价低 + 重文化 (人均 80-200)
│     └─ ✅ 老字号
├─ 价中 + 重社交 (人均 100-300)
│     └─ ✅ 网红 KOL
├─ 价低 + 重体验 (人均 30-80)
│     └─ ✅ 苍蝇馆
└─ 价高 + 重仪式 (人均 300+)
      └─ ✅ 私房菜
```

## ⚖️ 4 维度对比清单（发布文案必用）

| 维度 | 优势 | 不足 | 适合发布场景 |
|---|---|---|---|
| 老字号 | 文化 / 仪式 | 排队 / 价高 | 节假日大事件 / 文化深度选题 |
| 网红 | 出片 / 社交 | 价高 / 生命短 | 周五 / 周末 / 出片选题 |
| 苍蝇 | 烟火 / 实在 | 卫生 / 难找 | 反差选题 / 「本地人才知道」|
| 私房 | 高端 / 仪式 | 难预约 / 私密 | 商务 / 周年庆 / 高净值 |
