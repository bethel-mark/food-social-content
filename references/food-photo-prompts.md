# 食物摄影 AI 生图 Prompt 速查

> 美食与旅游生图的核心差异：**主体是食物而非风景**。本文档给出 8 套食物摄影专项 prompt 公式，对应 4 种食物场景。

---

## 🎯 通用公式（食物专属）

```
[菜品名 + 描述] + [构图：俯拍 / 45° / 侧面] + [光线 + 色调：暖光 / 蒸汽 / 油光泽] + [餐具 / 背景] + [风格参考] + [参数]
```

**关键 5 要素**：
1. **食品特写**（必须填具体菜名 + 形态）
2. **拍摄视角**（俯拍 / 45° 拍菜 / 平面排版）
3. **光线与色调**（暖光 / 自然光 / 蒸汽光）
4. **餐具背景**（木桌 / 砂锅 / 油布 / 报纸）
5. **风格锚点**（Bon Appétit 杂志 / Williams Sonoma 商业摄影）

## 🚫 食物摄影禁忌

- ❌ **模糊的「一堆菜」**（没有焦点）
- ❌ 极端俯拍（缺乏温度）
- ❌ **过曝白光**（餐厅最难看的光）
- ❌ 印 logo 的餐具特写（侵权）
- ❌ 包装袋 / 一次性餐具为主（显得廉价）
- ❌ AI 生成的「完美食物」与真实拍摄差距大（容易露馅）

## ✅ 食物摄影黄金法则

- ✅ **蒸汽 = 温度感**（最重要）
- ✅ **油光 = 新鲜**（最重要）
- ✅ **热气腾腾的画面** = 永远不出错
- ✅ 餐具反差（粗陶碗 + 精致的茶点）
- ✅ **颜色反差**（红色辣椒 + 绿色蔬菜 + 白色面皮）
